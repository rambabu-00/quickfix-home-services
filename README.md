# QuickFix Home Services

QuickFix Home Services is a web-based platform that connects customers with service providers such as plumbers, electricians, and carpenters.

## Features

- User Registration & Login
- Service Booking System
- Provider Dashboard
- Booking Management
- Responsive Design

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)
